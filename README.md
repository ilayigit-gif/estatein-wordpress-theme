# Estatein WordPress Theme

Custom responsive WordPress theme developed from the Estatein Figma design as part of a technical assessment project.

## Overview

This project was built using a custom WordPress theme structure with reusable templates, responsive layouts and custom frontend implementation based on the provided UI design.

## Features

- Fully responsive layout
- Custom WordPress theme development
- Mobile navigation menu
- Homepage sections
- Services page template
- Properties page template
- Contact page template
- FAQ accordion section
- FAQ Schema (JSON-LD)
- Property filter functionality
- Custom CSS animations and hover effects
- SEO-friendly structure

## Tech Stack

- WordPress
- PHP
- HTML5
- CSS3
- JavaScript
